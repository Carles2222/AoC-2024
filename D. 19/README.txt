El código de la Parte 1 evalúa si cada diseño de toalla puede formarse utilizando patrones disponibles. Utiliza un enfoque recursivo, combinando patrones desde una cadena vacía hasta coincidir con el diseño deseado. El código de la Parte 2expande esta lógica al contar todas las formas posibles de formar un diseño con los patrones dados, utilizando programación dinámica para optimizar el cálculo. Ambos programas verifican patrones desde una lista y determinan si las combinaciones cumplen con las especificaciones de los diseños deseados.
